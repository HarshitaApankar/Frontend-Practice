# Student Registration Form

This project is a simple **Student Registration Form** created using **HTML and CSS**. It is designed as a beginner-friendly frontend practice project to understand how forms are built and styled in web development.

The form collects basic student details such as full name, email, phone number, course, gender, date of birth, and address. It also demonstrates how different HTML form elements like input fields, dropdowns, text areas, and buttons can be used together in a structured layout.

## Features
- Collects student details in a structured form
- Includes different input types such as text, email, phone, and date
- Uses a dropdown for gender selection
- Includes a text area for address input
- Styled using CSS for a clean and simple user interface

## Technologies Used
- HTML
- CSS

## Purpose
This project was created for practicing frontend web development basics and improving understanding of HTML forms and CSS styling. It is suitable for beginners who are learning how to create interactive webpage forms.
