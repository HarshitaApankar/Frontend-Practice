# Profile Card

## 📌 Project Overview

This project is a simple **Profile Card** created using **HTML** and **CSS**. It displays a user's name and a short introduction in a clean, centered card layout. The design uses soft colors, rounded corners, and a responsive flexbox layout to create an attractive user interface.

## ✨ Features

* Clean and minimal user interface
* Centered profile card using CSS Flexbox
* Responsive layout
* Rounded card design
* Soft pastel background
* Easy to customize with your own information

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📂 Project Structure

```
Profile-Card/
│
├── index.html
└── README.md
```

## 🚀 How to Run the Project

1. Download or clone the project.
2. Open the project folder.
3. Double-click the `index.html` file, or open it in any web browser.
4. The profile card will be displayed in the center of the page.

## 🎨 Design Details

* **Background Color:** Light Purple (`rgb(237, 205, 253)`)
* **Card Background:** White
* **Heading Color:** Purple (`rgb(106, 22, 162)`)
* **Layout:** CSS Flexbox
* **Border Radius:** 12px
